x4Pay Token

Contract Address: 6ikxp2KnQcdCik8Aadi2ScE7qgH4j5C7BvSyA29Wpump

Creates a decentralized trust layer for autonomous machine networks, enabling verifiable machine registration and reputation-based verification levels through cryptographic proof mechanisms.

x4 Core Registration

Machine operators must burn tokens permanently to register their x4 Core machines on the network.

• Tokens are permanently destroyed (burned) during registration
• One-time cost per machine registration
• Unregistered machines show warning to users
• Registration cost may increase with network growth

x4 Levels

Users and operators achieve higher x4 levels by holding certain amounts of x4Pay tokens in their wallet.

• Tokens must be held (not burned) to maintain level
• Registered machines to only upgrade levels
• Level displayed as verification badge to users
• Dynamic system - level changes with token balance

Token Usage Flow

1. Registration Process

Machine operator burns required tokens to register x4 Core machine.

Result: Machine appears as "Registered" in user applications and agents looking for machines

2. Level Achievement

Operator maintains token balance to display higher x4 level badge.

Result: x4 levels displayed on every tap and connect from mobile and web app and agent discovery
